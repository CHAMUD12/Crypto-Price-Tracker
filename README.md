# Installation Commands

```sh
# Install core dependencies for state management, API calls, and charts
npm install react-redux @reduxjs/toolkit axios recharts @tanstack/react-query

# Install Tailwind CSS and required plugins
npm install -D tailwindcss postcss autoprefixer

# Install Recharts for data visualization
npm install recharts

# Install Zustand for state management (optional - not)
npm install zustand

# Install React Router DOM for navigation (optional - not)
npm install react-router-dom

# Install Redux Persist for persisting Redux state
npm install redux-persist
